ReplyMate Privacy Policy

Last updated: 2026

1. Information We Collect

ReplyMate generates an anonymous user ID to manage usage and subscriptions.

2. Email Processing

Email content is processed temporarily to generate AI replies and is not stored.

3. Third-Party Services

ReplyMate uses OpenAI for AI reply generation and Stripe for payment processing.

4. Data Storage

ReplyMate stores anonymous usage data and subscription status.

5. Security

All communications are encrypted using HTTPS.
